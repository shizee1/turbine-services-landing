# TurbineServices.Experts — Landing Page

A fully responsive landing page for a gas turbine parts supply company. Built with clean HTML and CSS from a Figma design.

## 🚀 Live Demo
[View the site](https://heartfelt-brioche-2d7026.netlify.app)


## 🛠️ Tech Stack
- **HTML5** — Semantic markup
- **CSS3** — Flexbox, Grid, Custom Properties
- **BEM** — Block-Element-Modifier naming
- **Figma** — Pixel-perfect layout implementation

## 📱 Responsive Breakpoints
- Desktop: 1920px (full layout)

## 🎯 Key Features
- **Hero section** with background overlay and search
- **About Us** block with image overlay statistic (5M+)
- **Advantages** cards with icons
- **Markets** section with client list and CTA
- **Products** grid (15 items with icons)
- **Google Reviews** with rating widget
- **Contact form** with Google Maps embed
- **Footer** with 4 columns + copyright bar

## 🧠 What I Learned
- Working with Figma designs
- Implementing BEM methodology
- Building complex grid layouts
- Creating overlay effects
- Embedding Google Maps iframe
- Deploying to Netlify from GitHub
